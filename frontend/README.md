# SoundTouchBridge Frontend

React + Vite SPA für die SoundTouchBridge Web-UI.

## Installation

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

UI läuft auf: http://localhost:3000

API-Calls werden zu http://localhost:8000 proxied (siehe vite.config.js).

## Build

```bash
npm run build
```

Build-Output: `dist/`
